## Overview

This repository contains materials for the course, including:

- R scripts from course lectures  
- Capstone project outputs from each student team  
- SDRF files and associated metadata  
- Optional downstream analysis results and reports  

---

## Repository Structure

Each team has a dedicated folder containing:

- `SDRF.tsv` — Sample and Data Relationship Format file  
- `identification/` — Protein/precursor identification results  
- `optional_analysis/` — Statistical and downstream analyses
